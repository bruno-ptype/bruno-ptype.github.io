HTML


HTML es el lenguaje de marcado esencial para la creación de contenido web. Define la estructura básica de las páginas web mediante el uso de etiquetas, permitiendo la creación de documentos desde simples hasta complejas aplicaciones interactivas



Internet como tecnología


Internet es una red informática descentralizada que permite la conexión entre computadoras a través de un protocolo de comunicaciones. Es una red global que comparte contenidos y no tiene una ubicación física específica, ya que está presente donde haya computadoras conectadas.



Protocolo HTTP (Hypertext Transfer Protocol)


HTTP es el protocolo que permite la petición y transferencia de datos y recursos en la web, como documentos HTML. Funciona bajo una estructura cliente-servidor, donde el cliente (normalmente un navegador web) inicia las solicitudes de datos.


Arquitectura Cliente-Servidor


Es un modelo arquitectónico distribuido compuesto por un servidor que proporciona servicios o recursos, y múltiples clientes que consumen estos servicios. El cliente actúa como una interfaz para representar los datos y el servidor realiza el procesamiento principal.
